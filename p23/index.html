Задача

Работа с https://pokeapi.co/

    Реализовать
  • 1. При входе на страницу пользователь должен выдеть список с 10 покемонов с краткой информацией
  • 2. При клике на покемона список должен пропасть и при этом отобразиться подробная ифнормация по покемону
  • 3. При обновлении страници видя подробному информацию покемону она должна остаться
  • 4. При клике на кнопку назад подробная информация пропадает и снова видно список покемонов
  • 5. Покрыть тестами

Для выполнения пункта 3, обычно используються query параметры в url. Но в чистом виде, изменения query параметров приводят к перезагрузке страницы.
По этому желательно использовать хеш
