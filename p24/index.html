Переменные let и const

В ES2015 вместо var введены новые переменные let и сonst - константа
Отличия let:

  • Область видимости переменной.
  • Переменная начинает существовать только после объявления.
  • Для каждой итерации в цикле создаётся своя переменная.

Создать переменные с помощью ключевых слов let и const и протестировать их свойства.
