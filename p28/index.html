Web Workers

Web Workers предоставляют простое средство для запуска скриптов в фоновом потоке.
new Worker("file.js") / new SharedWorker("file.js") - доступен нескольким скриптам.
Методы:

  • postMessage(data)
  • onmessage(f)
  • terminate()
  • close()
  • importScripts()

History API

DOM window объект предоставляет доступ к истории браузера через объект history.
Методы:

  • back()
  • forward()
  • go(n), где n - относительный показатель к текущей странице
  • pushState(stateObj, "title", "url")
  • replaceState(stateObj, "title", "url")

Свойства:

  • history.length
  • history.state
Событие:
  • popstate

Geolocation

Геолокация позволяет использовать место положения пользователя в веб-приложениях.
API геолокации доступено через navigator.geolocation.
Методы:

  • getCurrentPosition(success, error)
  • watchPosition( function(position){}, error, options) -> watchID, где position содержит свойства coords.latitude и coords.longitude
  • clearWatch(watchID)
Вывести координаты своего местоположения на странице.

Touch events

Touch-события предлагают возможность интерпретации воздействия пальца (или стилуса) на тач-экраны или трекболы.
Интерфейсы:

  • TouchEvent - событие которое происходит когда изменяется состояние касания на поверхности.
  • Touch - одна точка контакта между пользователем и поверхностью касания.
  • TouchList - группа касаний; это используется когда пользователь, например, касается поверхности несколькими пальцами одновременно.

События для TouchEvent:

  • touchstart
  • touchend
  • touchmove
  • touchcancel
Свойства:
  • changedTouches -> TouchList
  • targetTouches -> TouchList
  • touches -> TouchList
  • altKey, ctrlKey, shiftKey, metaKey

Drag and drop

Позволяет пользователю нажать и удерживая зажатой кнопку мыши над элементом, переместить его на другую позицию, отпустив кнопку мыши пользователь может оставить элемент на новой позиции.
Этапы:

  • Определить переносимый элемент.
  • Определить данные, которые могут быть перемещены.
  • Определить возможные эффекты переноса.
  • Определить область назначения.
  • Определить действие на drop.

События:

  • dragstart
  • dragenter
  • dragover
  • dragleave
  • drag
  • drop
  • dragend
Пример.

event.dataTransfer.effectAllowed = "copy";
event.dataTransfer.dropEffect = "copy";
Эффекты:

  • none
  • copy
  • move
  • link
  • copyMove
  • copyLink
  • linkMove
  • all (copy, move, link)
